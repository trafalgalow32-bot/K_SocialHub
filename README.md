# K_SocialHub
# 🌐 K_SocialHub: Science & Technology for Social Problem Soving

## 📝 프로젝트 개요 (Introduction)
본 프로젝트는 **과학기술 기반 사회문제해결 R&D**의 성과를 가시화하고, 복잡한 사회문제를 데이터 관점에서 분석하기 위한 마중물 프로젝트입니다. 

본 프로젝트에서 제시하는 자료는 KISTEP(한국과학기술기획평가원)의 공공누리집 문서들을 참고하여 출처 표기 후 활용하고 있습니다.

## 🎯 주요 타겟: 전력 수급 (Energy Power Supply)
현대 사회의 핵심 인프라이자 AI 시대의 근간인 **전력 수급** 문제를 첫 번째 분석 과제로 선정하였습니다.
- **Why?** AI 데이터 센터 증설 및 에너지 전환 이슈와 맞물려, 전력 수급 안정성은 단순한 인프라를 넘어 국가적 사회문제로 대두되고 있습니다.
- **Goal:** 뉴스 데이터(Big Kinds 등)와 공공데이터를 결합하여 전력 수급 이슈의 흐름을 파악하는 인사이트 도구 개발.

## 🛠 Tech Stack (예정)
- **Language:** Java, Python
- **Frontend:** HTML5, CSS3, JavaScript
- **Data Source:** Big Kinds (뉴스 데이터 분석), 공공데이터포털
